# React SEO Practice - React Helmet

一個 SEO 套件 **React Helmet** 基於 [Create React App](https://github.com/facebook/create-react-app) 建立。

該專案用來練習 **React Helmet**，基本上專案內容沒有任何變動。

## 使用方法

```
1. $ git clone xxx

2. $ cd react-seo-helment

3. $ npm i / yarn i

4. $ npm run / yarn start 擇一

5. Enjoy it!
```
